About BookHub

BookHub is a simple, responsive, and API-powered book explorer. It uses the public Open Library API to fetch real book data and lets you search and curate your personal favorites list stored locally in your browser.

Features

  • Browse books fetched from the Open Library API
  • Search by keywords or titles
  • Add or remove favorites (persisted with localStorage)
  • Navigate across Home, Favorites, and About pages
  • Responsive layout built with Tailwind CSS

Tech Stack

  • HTML5 + Vanilla JavaScript (ES Modules)
  • Open Library Search API
  • Tailwind CSS via CDN
  • localStorage for favorites
